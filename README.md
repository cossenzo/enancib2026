# enancib2026
Artefatos e materiais suplementares do artigo submetido ao Enancib 2026 - GT 2:
- Ontologia DeFiSocialOnto no formato RDF/XML: `DeFiSocialOnto.owl`
- Taxonomia experimental e _corpus_ piloto: `graph_dataset.xlsx`
- Script Python para criação de triplas RDF a partir do _dataset_: `create_defisocialonto_graph_triples.py`
- Triplas RDF para criação do grafo de conhecimento no GraphDB: `defisocialonto_graph_dataset_instances.ttl`
- Consultas SPARQL executadas para respostas às questões de competência formuladas: `sparql_queries_competency_questions.md`
